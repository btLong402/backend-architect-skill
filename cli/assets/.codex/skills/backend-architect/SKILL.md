# Backend Architect Skill

Search the backend architecture knowledge base.

## Usage

```bash
python3 .codex/skills/backend-architect/scripts/search.py "<query>" --domain <domain>
```

**Domains**: architecture, database, security, product, language, api, naming, error, platform
**Stacks**: go, python, node, java, dotnet, rust

## Examples

```bash
python3 .codex/skills/backend-architect/scripts/search.py "ecommerce" --domain product
python3 .codex/skills/backend-architect/scripts/search.py "microservices" --domain architecture
python3 .codex/skills/backend-architect/scripts/search.py "orm" --stack python
```
